### MCP Servers to Activate
- [ ] github — GitHub access for PRs, issues, CI status, and code review from inside Claude. Add the GitHub MCP server (claude.ai connectors, or `claude mcp add`) and authenticate against the tiborpilz org.
- [ ] Claude Code Remote — cloud/web Claude Code sessions: scheduled triggers, PR watching, and remote environments. Available when you run Claude Code on the web; sign in with your Claude account.

### Skills to Know About
